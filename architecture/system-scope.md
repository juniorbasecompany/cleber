# Escopo do sistema

## O que o projeto constrói

O projeto está construindo uma base comum para sair de planilha operacional e chegar a um sistema capaz de:

- planejar a operação;
- registrar o realizado;
- comparar resultado esperado e observado;
- simular cenário;
- consolidar resultado por recorte operacional e período;
- sustentar visão técnica, econômica e gerencial sobre a mesma base.

## Prioridade atual

- prioridade funcional atual no domínio avícola;
- desenho estrutural preparado para mais de um nicho;
- preservação do conhecimento hoje espalhado em planilha;
- evolução gradual por camada, sem tentar fechar todo o produto de uma vez.

## O que não pertence aqui
