# Evento agrícola

Família típica de evento governado:

- plantio;
- replantio;
- aplicação ou manejo relevante;
- ajuste de área;
- ajuste de população;
- atualização de perda esperada;
- colheita parcial;
- encerramento de segmento ou de ciclo.

O evento deve permanecer explícito, datado, auditável e reconstruível.
