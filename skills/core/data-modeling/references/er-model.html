Modelo ER inicial para a skill data-modeling

Este artefato representa um modelo relacional base para o sistema com separação entre núcleo estrutural, metadado governado, fato diário, localização e apresentação. O foco é clareza arquitetural, rastreabilidade, vigência e adaptação a mais de um nicho sem contaminar o núcleo.

Grão oficial
Fato diário
Semana, mês e período são derivados da mesma base diária.
Escopo
Hierarquia única
`country` entra como nó opcional de `location_node`.
Financeiro
Moeda local auditável
Conversão cambial existe apenas em consulta e relatório.

Legenda rápida

Cada caixa abaixo é uma tabela. Os badges destacam chaves e tipos especiais.

PK chave primária
FK chave estrangeira
UQ unicidade / técnica
JSONB estrutura flexível governada

Tipos sugeridos seguem PostgreSQL. Os exemplos mostram valores curtos para facilitar leitura, não o conjunto completo de regras.

Mapa ER resumido

Visão de alto nível das dependências principais entre identidade, metadado, fato, proveniência e camada de apresentação.
