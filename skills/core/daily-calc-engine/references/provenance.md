# Proveniência

Cada fato diário deve ser rastreável até:

- evento ou entrada de origem;
- curva ou parâmetro utilizado;
- versão da curva, quando houver curva;
- vigência da curva, quando houver curva;
- escopo que forneceu o valor;
- fórmula ou regra aplicada;
- versão de cálculo;
- timestamp de execução.

Explicabilidade é obrigatória para garantir confiança operacional.
