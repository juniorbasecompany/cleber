# Attribute Contract

Each configurable attribute should define at least:

- display name
- machine key
- value type
- business role in the calculation
- granularity
- allowed scope
- validity behavior
- allowed origins
- unit and precision
- aggregation rule
- optional formula or curve reference
- visibility in analytics and panels
- optional localized labels by locale
- optional country-level override behavior when the attribute participates in fallback
