# Catálogo de regra

Família de regra governada inclui:

- resolução por escopo;
- derivação diária;
- busca de curva;
- conversão econômica;
- restrição de capacidade;
- classificação de qualidade;
- sobrescrita de cenário.

Cada regra deve definir:

- propósito;
- entrada;
- saída;
- escopo permitido;
- origem permitida;
- requisito de explicabilidade.
